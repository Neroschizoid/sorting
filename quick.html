Quick Sort

"Quick Sort is a divide-and-conquer algorithm that selects a 'pivot' element and partitions the array into two halves, sorting each half recursively."

"This algorithm is generally faster than Merge Sort for smaller datasets, with an average time complexity of O(n log n), but can degrade to O(n^2) in the worst case."

"Quick Sort is not a stable sorting algorithm, but it is in-place, requiring minimal additional storage."
